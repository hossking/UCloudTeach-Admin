### 使用说明

1.  克隆本仓库：`git clone https://gitee.com/h0ss/ucloud-teach-admin.git`
2.  安装依赖：`npm install`
3.  运行项目：`yarn run serve-dev`
4.  确保后端项目已启动，修改`.env.dev`文件中的地址与端口信息
5.  本地构建请在浏览器访问`http://localhost:8080`
